# Marketplace - Detail dodavatele (/marketplace/vendor/[id])

## Přehled modulů a prvků

### 1. HERO SEKCE / HEADER
**Základní prvky:**
- Cover photo (velký banner obrázek)
- Logo dodavatele
- Název firmy
- Kategorie (badge)
- Lokace (město, region)
- Hodnocení (hvězdičky + počet recenzí)
- CTA tlačítko "Kontaktovat"
- Tlačítko "Přidat k oblíbeným"

**Možnosti rozšíření:**
- Badge "Ověřený dodavatel"
- Badge "Premium"
- Badge "Rychlá odpověď"
- Dostupnost (volné termíny)
- Počet dokončených svateb
- Social media odkazy
- Tlačítko "Sdílet"
- Video prezentace (autoplay muted)
- Carousel více cover fotek

---

### 2. NAVIGACE / TABS
**Základní prvky:**
- Přehled
- Galerie
- Služby & Ceny
- Recenze
- Kontakt

**Možnosti rozšíření:**
- Portfolio
- Blog / Články
- FAQ
- O nás / Náš příběh
- Tým
- Ocenění
- Real weddings (realizované svatby)

---

### 3. PŘEHLED (Overview)
**Základní prvky:**
- Krátký popis (elevator pitch)
- Dlouhý popis firmy
- Klíčové informace:
  - Založeno (rok)
  - Počet svateb
  - Styl (moderní, vintage, boho...)
  - Cenové rozpětí
  - Lokace působení
- Kontaktní tlačítko

**Možnosti rozšíření:**
- Video představení
- Highlights / USP (unikátní výhody)
- Certifikace a ocenění
- Členství v asociacích
- Jazyky komunikace
- Timeline dostupnosti
- Quick facts (ikony + text)
- Featured v médiích

---

### 4. GALERIE
**Základní prvky:**
- Grid layout (3-4 sloupce)
- Lightbox pro zobrazení
- Navigace mezi obrázky
- Počet fotek

**Možnosti rozšíření:**
- Kategorie fotek (svatby, detaily, místa, produkty)
- Filtrování podle stylu
- Video galerie
- 360° fotky
- Before/After slider
- Pinterest-style masonry layout
- Download možnost (pro klienty)
- EXIF data (pro fotografy)
- Tagy a popisky
- Instagram integrace

---

### 5. SLUŽBY & CENY
**Základní prvky:**
- Seznam služeb
- Popis každé služby
- Cenové rozpětí nebo konkrétní ceny
- Co je zahrnuto v ceně

**Možnosti rozšíření:**
- Balíčky (Basic, Standard, Premium)
- Srovnávací tabulka balíčků
- Přídavné služby (add-ons)
- Sezónní ceny
- Slevy a akce
- Kalkulačka ceny
- Tlačítko "Vyžádat nabídku"
- PDF ke stažení (ceník)
- Platební podmínky
- Storno podmínky
- Dostupnost termínů (kalendář)

---

### 6. RECENZE & HODNOCENÍ
**Základní prvky:**
- Celkové hodnocení (hvězdičky)
- Počet recenzí
- Seznam recenzí:
  - Jméno klienta
  - Datum svatby
  - Hodnocení (hvězdičky)
  - Text recenze
  - Datum přidání

**Možnosti rozšíření:**
- Filtrování recenzí (nejnovější, nejvyšší hodnocení)
- Kategorie hodnocení:
  - Kvalita služeb
  - Komunikace
  - Profesionalita
  - Poměr cena/výkon
  - Dodržení termínů
- Fotky od klientů
- Video testimonials
- Odpovědi dodavatele na recenze
- Ověřené recenze (badge)
- Google Reviews integrace
- Facebook Reviews integrace
- Statistiky (% doporučení)
- Helpful tlačítko (bylo užitečné?)

---

## TECHNICKÉ SPECIFIKACE

### URL struktura:
- `/marketplace/vendor/[id]` nebo
- `/marketplace/vendor/[slug]` (SEO friendly)

### Responzivita:
- Mobile: single column, sticky CTA button
- Tablet: 2 columns (content + sidebar)
- Desktop: 3 columns možnost

### Performance:
- Lazy loading obrázků a videí
- Progressive image loading
- Optimalizované thumbnaily
- CDN pro media

### SEO:
- Strukturovaná data (LocalBusiness, Service, Review schema)
- Open Graph tagy (sdílení na social media)
- Meta description unikátní pro každého dodavatele
- Breadcrumbs
- Alt texty u obrázků

### Analytics:
- Tracking zobrazení profilu
- Tracking kliknutí na kontakt
- Tracking času na stránce
- Heatmapy (kam uživatelé klikají)

### Integrace:
- Firebase Firestore (data dodavatele)
- Google Maps API
- Email service (kontaktní formulář)
- Calendar API (booking)
- Social media APIs (Instagram, Facebook)
- Review platforms (Google, Facebook)
